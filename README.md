## 🚀 Featured Projects

### 🔐 [Dead Man's Switch](https://github.com/mounir-nawwar/Dead-Mans-Switch)
**Cryptographically secure delayed message delivery**  
`TypeScript` `Node.js` `PostgreSQL` `Docker` `React` `Web Crypto API`

- Zero-knowledge architecture — server stores only ciphertext, plaintext never leaves the browser
- AES-256-GCM encryption entirely in-browser via Web Crypto API
- Shamir's Secret Sharing (2-of-3 threshold) built from scratch using Lagrange interpolation over GF(2⁵²¹−1) — no third-party crypto libraries
- Distributed backend: two isolated Node.js/Express/TypeScript services + two independent PostgreSQL instances via Docker Compose
- JWT auth with bcrypt (12 rounds), timing-attack-resistant login, replay attack prevention
